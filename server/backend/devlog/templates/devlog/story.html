Why We Built This

As final year ECE students, we often noticed the lack of a centralized platform where learning resources, exams, and mentor guidance could exist in one place. Our LMS aims to bridge that gap — creating a unified space for students and mentors to collaborate, share, and grow academically.

Core Features at a Glance

📚 Course Management

Admins can create, organize, and publish course materials with ordered content and type-specific validations.

📝 Exam System

Dynamic exams with scoring, question banks, and result tracking capabilities.

👤 Auth System

JWT-based role-specific login and access control for admins, mentors, and students.

📰 Newsfeed

A simple yet effective community board where mentors can post updates and interact with learners.

Challenges & Solutions

  • Data Management: Structuring course content, user roles, and exams required thoughtful database design — solved through normalized PostgreSQL schemas and Django models.
  • Frontend-Backend Integration: Managing React and Django on separate servers posed CORS issues, solved using proper configuration in Django REST framework.
  • Admin Privileges: We needed a dedicated admin panel with restricted views — handled using custom permissions and route guards.
  • Team Collaboration: Coordinating tasks among three developers — resolved using Git branches and GitHub Projects for organized collaboration.

What’s Next

  • 📊 Student dashboard for visualizing course progress
  • 📧 Email notifications and alerts for new content
  • 🔗 Integration of live class APIs (like Zoom/Google Meet)
  • 📦 LMS mobile app with push notification support
